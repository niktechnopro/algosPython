Introduction

The following describes how to use the Cisco IOS command-line interface (CLI) to perform a basic software configuration for your router or switch.

How to change the Hostname

The hostname is used in CLI prompts and default configuration filenames. If you do not configure the router hostname, the router uses the factory-assigned default hostname “Router.”

Do not expect capitalization and lowercasing to be preserved in the hostname. Uppercase and lowercase characters are treated as identical by many Internet software applications. It may seem appropriate to capitalize a name as you would ordinarily do, but conventions dictate that computer names appear in all lowercase characters. For more information, see RFC 1178, Choosing a Name for Your Computer.

Summary Steps:

Command Command Line Description
1.enable switch>enable this will put you into priviledge EXEC mode
2.configure terminal switch#configure terminal enters global configuration mode
3.hostname <name> switch(config)#hostname my_new_switch where <my_new_switch> is a new hostname
4.get out my_new_switch(config)#end exit to EXEC(privilege) mode
  1. 'hostname' name - how you want to name the divece
  2. Verify that the router prompt displays your new hostname
  3. 'end' - Ends your configuration session by exiting to EXEC(privilege) mode
How to set up MOTD

To configure the message-of-the-day (MOTD) banner that displays when the user logs in to a Cisco switch/router, use the banner motd command. To revert to the default, use the no form of this command.

Summary Steps:

Command Command Line Description
1.enable switch>enable this will put you into priviledge EXEC mode
2.configure terminal switch#configure terminal enters global configuration mode
3.banner motd <delimiter> message <delimiter> switch(config)#banner motd #Welcome# where #-indicates the start and end of the message
4.get out switch(config)#end exit to EXEC(privilege) mode
